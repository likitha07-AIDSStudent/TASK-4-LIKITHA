# Decode Labs Internship – Project 4: Data Visualization

## Project Overview
This project focuses on data visualization using Python. Different charts were created to analyze an e-commerce dataset and extract meaningful business insights.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
- Cleaned E-commerce Dataset (.xlsx)

## Visualizations
1. Most Ordered Products (Bar Chart)
2. Payment Method Distribution (Pie Chart)
3. Order Status Distribution (Pie Chart)
4. Referral Source Analysis (Bar Chart)
5. Distribution of Total Price (Histogram)
6. Quantity vs Total Price (Scatter Plot)
7. Total Price Distribution (Box Plot)
8. Sales Trend Over Time (Line Chart)

## Key Insights
- Identified the most ordered products.
- Analyzed customer payment preferences.
- Studied order status distribution.
- Compared referral sources.
- Examined the distribution of order values.
- Explored the relationship between quantity and total price.
- Detected outliers using a box plot.
- Observed sales trends over time.
